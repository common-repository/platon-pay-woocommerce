The module for receiving payments from cards for online stores and landing pages.

== Description ==

«Platon Pay WooCommerce» is an ideal solution for online stores operating on the WooCommerce platform for receiving payments from cards (visa, mastercard, prostir, apple pay, google pay, visachekout, masterpass) and Privat24.

Easy and convenient installation and configuration of the module will allow you to set up the ability to accept card payments on your website in a few minutes.

«Platon Pay WooCommerce» is perfect for online stores operating on the WooCommerce platform.

Platon Company (https://platon.ua) - tried to take into account all the wishes of customers and developed the easiest to install and configure module for receiving payments from cards for online stores.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the CMS-> «Platon Pay WooCommerce» screen to configure the plugin
4. After activation:

You need to contact the specialists of our company and get a consultation and a data set for further setting up the plugin discounts.

Url - Url issued by platon.ua for sending a POST payment request.

Secret key (ID) - The key issued by platon.ua to identify the Client (available after registering the merchant in the platon.ua system).

Password - Password provided by platon.ua involved in the formation of the MD5 signature (available after registering the merchant in the platon.ua system).

Callback Url - Report to Tech. platon.ua support your Callback Url (POST will be sent to it answers from the platon.ua system - about a successful transaction).

Technical support contacts: https://platon.ua/contacts

Apply for connection: https://platon.ua/payment-request
